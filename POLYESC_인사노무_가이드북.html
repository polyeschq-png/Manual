이 가이드북에 대하여

본 자료는 폴리 캠퍼스 원장·관리자의 일상 운영 실무를 정리한 매뉴얼입니다. CAPS 근태 관리와 매월 근태 제출, 채용 시기·공고, 복리후생 제도(수강료·경조사·항공권 등)까지 담고 있습니다.

왼쪽 목차에서 원하는 항목으로 바로 이동할 수 있습니다. 향후 다른 교육 자료도 이 책에 이어서 추가·운용됩니다.

01
CHAPTER 01

캠퍼스 근태 관리

1-1

근태 아이디 생성

  1. 입사자 CAPS ID 생성
    • 관리자 : 원장, 부원장, 교수부장(KC·EC·AC), 상담실장, 유치부 주임
    • 관리자 외 : 유치부 교사, 상담교사, 조교, 사서, 안전선생님 등
  2. 관리자 그룹웨어 연동
    • 본사로 그룹웨어 연동 신청 — 미연동 시 근태 관리 불가
1-2

퇴사자 아이디 삭제

  1. 퇴사자 CAPS ID 삭제
    • 출입이 불가하도록 퇴사 다음날 바로 삭제 필요
1-3

근태 관리

  1. 매달 10일 이전 근태 자료 본사로 제출
    • 대상 기간 : 전달 1일부터 말일
    • 수신인 : 이윤지 대리 (rachel.lee95@esc.edu-poly.com)
  2. 근태 조정
    • 캠퍼스 모든 인원의 출퇴근이 잘 기록되었는지 확인
    • 특이사항 조정 (예: 셔틀탑승, 외근 등)
    • CCTV 증빙 자료 취합
  3. 근태 확인 리스트
    • 경로 : 그룹웨어 > 인사/근태 > 근태신청관리 > 근태현황관리
근태 조정 사유 유형
  • 셔틀탑승
  • 연차 (경조사, 개인연차 등)
  • 출퇴근 기록 누락
02
CHAPTER 02

채용

2-1

채용 시기

① 상반기 (봄학기 시작)

  • 일반 근로자 : 12월부터 채용 시작
  • 외국인 근로자 : 11월부터 채용 시작

② 하반기 (가을학기 시작)

  • 일반 근로자 : 6월부터 채용 시작
  • 외국인 근로자 : 5월부터 채용 시작
포인트

외국인 근로자는 비자 등 절차가 있어 일반 근로자보다 한 달 먼저 채용을 시작합니다.

2-2

채용 공고

  • 모든 폴리 직영 통일 공고 디자인 사용
    • 매 상/하반기 채용공고 디자인 변경
    • 인사팀으로 공고 업로드 요청 (단시간 근로자 채용공고는 캠퍼스에서 업로드)
2-3

인원 현황

  • 매주 인원현황 확인
    • 인사팀에서 인원현황 확인 요청

3-4

자녀 입학 축하선물

  • 대상 : 재직 중인 정규직 임직원 (단, 휴직자는 제외)
  • 지원 : 초·중·고·대학교 입학 시 20만원 상품권 (현금 지급 불가)
  • 증빙 : 자녀 입학증명서 및 가족관계증명서 (재학증명서 대체 가능)
  • 신청 경로 : 그룹웨어 기안서 상신(증빙 첨부 必) → KCMS 지출 상신 → 상품권 20만원 별도 구매 → 대상자 지급
3-5

자녀 출산 축하선물

  • 대상 : 재직 중인 정규직 임직원
  • 지원 : 현금 10만 원
  • 신청 경로 : 그룹웨어 기안서 상신 → KCMS 지출 상신 → 캠퍼스 별도 지급
3-6

거주 지원 _ 외국국적자

  • 대상 : 교포/원어민 수업 및 관리 가능한 Full Time 부원장 / 교수부장 / 원어민강사
  • 지원내용
    • 보증금 : 주택보증금 한도 500만원, 월세 60만원 사택 지원
    • 보조금으로 대체 시, 월 40만원 (근로소득 과세 처리)
3-7

항공권 지원 _ 해외체류자 입/출국

1. 입사 및 퇴사 시 항공료 지원

  • 지원 대상 : 해외에서 입사하거나 퇴사하는 교포/원어민 등 해외체류자
  • 지원 조건
    • 출국 항공권의 도착지(도시) = 입국 항공권의 출발지(도시)와 동일해야 함
    • 입사 시 입국 항공료, 퇴사 시 출국 항공료 모두 지원 가능
    • 입국 항공료 : 입사 후 ARC 및 국내 계좌 발급 완료 후 지급 (국외 계좌 지급 불가)
    • 출국 항공료 : 퇴사 시점에 맞춰 지급

2. 계약기간 6개월 미만 시 항공료 처리

  • 입사 첫 계약기간이 6개월 미만 근무할 경우 : 입국 시 회사가 제공한 항공료는 미근무일수만큼 환급 (급여에서 공제)
  • 출국 항공료는 계약 만료 시에만 지원 — 계약 조기 종료 시 퇴사 항공료는 지원되지 않음

3. 지급 처리 방법 (캠퍼스 실비 처리 기준)

  • 항공료는 캠퍼스에서 실비 정산 처리 (KCMS 지출 상신 必)
    • 법인 발권 시 : 실비 청구 서류(항공권 영수증) 제출 필요 — KCMS 지출 상신 시 첨부
    • 개인 발권 시 : 지출 상신 방법은 위와 동일 → 대상자 급여 계좌 지급 처리
